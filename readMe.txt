========Team Rankings HtmlAgilityPack Table Scraper========


To use, make sure you are using the HtmlAgilityPack, if not you can get the package through nuGet manager.

Replace the url in the var getpage variable in line8 with the url of the teamrankings page containing
your desired table. 

The program should scrape the table into a dataTable then sort the table alphabetically by team name.
